Accessibility Statement

SecTech is committed to providing a website that is accessible to the widest possible audience, regardless of circumstance and ability.

Commitment to Accessibility

We aim to adhere as closely as possible to the Web Content Accessibility Guidelines (WCAG 2.2), published by the World Wide Web Consortium (W3C). These guidelines explain how to make Web content more accessible for people with disabilities. We believe that accessibility is a fundamental right and a responsibility.

Measures Supported

SecTech takes the following measures to ensure accessibility of our website:

  • Include accessibility as part of our internal policies.
  • Integrate accessibility into our procurement practices.
  • Provide continual accessibility training for our staff.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. The SecTech website is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Feedback

We welcome your feedback on the accessibility of the SecTech website. Please let us know if you encounter accessibility barriers on our site.

Compatibility with Browsers and Assistive Technology

The SecTech website is designed to be compatible with the following assistive technologies:

  • Recent versions of screen readers (NVDA, JAWS, VoiceOver)
  • Standard browser magnification
  • Keyboard navigation

Technical Specifications

Accessibility of the SecTech website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS
