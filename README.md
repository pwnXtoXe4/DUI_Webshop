# Webshop - SecTech

Webshop for 1st practical assignment in "Data and IT Law" @ University of Technology Vienna

## Project Structure

- `frontend/` - Frontend code
- `data/` - Data files
- `docs/` - Documentation
- `README.md` - This file

Note that there is no backend code, as the assignment only required a static website.

## Company: SecTech GmbH.

`SecTech GmbH.` is a company that sells security software products. Products include:

- Antivirus software
- Firewall software
- VPN software
- Password manager software

Products are targeted towards companies with a licensing model.
Licenses can be bought for single products or for a bundle of products.
